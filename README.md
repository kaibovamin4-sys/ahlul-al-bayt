# Ahlul al-Bayt — Цифровой Музей Знаний

Премиальная editorial-платформа о шиитском исламе. Цифровой архив знаний об Ахль аль-Байт: 12 Имамов, дуа и зияраты, Коран и тафсиры, исламский календарь и время намаза.

## Возможности

- **12 Имамов** — биографии с арабскими именами и историческим контекстом
- **Дуа и зияраты** — полные тексты в три колонки: арабский, транскрипция, перевод
- **Исламский календарь** — интерактивный, с событиями по всем 12 месяцам
- **Время намаза** — джафаритский расчёт для любого города мира (Aladhan API)
- **Марджа ат-таклид** — биографии великих аятолл
- **Библиотека знаний** — категории источников
- Светлая и тёмная темы, полная адаптивность

## Запуск локально

```bash
python3 -m http.server 8000
```

Открыть http://localhost:8000

> Нужен именно HTTP-сервер, не `file://` — иначе `duas.json` не загрузится из-за CORS.

## Добавление нового дуа

1. Открыть `data/duas.json`
2. Добавить новый ключ:

```json
"ashura": {
  "label": "Зиярат · Ашура",
  "title": "Зиярат Ашура",
  "arabic_title": "زِيَارَة عَاشُورَاء",
  "desc": "Описание...",
  "verses": [
    { "section": "Название раздела" },
    { "ar": "…", "tr": "…", "ru": "…" }
  ]
}
```

3. В `index.html` у соответствующей карточки заменить «Скоро →» на «Читать →»

Ключ в JSON должен совпадать с аргументом в `onclick="openDua('ashura')"`.

## API

| Сервис | Назначение |
|---|---|
| [Aladhan](https://aladhan.com/prayer-times-api) | Время намаза, `method=0` — джафаритский расчёт |
| [Nominatim](https://nominatim.org/) | Геокодинг городов |

- **Акцент** — золото `#c9a84c`
- **Фон** — тёмный `#0a0c0b` / светлый `#f5f2eb`
